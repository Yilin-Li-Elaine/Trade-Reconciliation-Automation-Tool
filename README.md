# Trade Reconciliation Automation Tool (Excel VBA)
## Overview
This project is an automated trade reconciliation tool built in Excel VBA.
It compares Internal and Broker trade records and identifies discrepancies.
## Features
* Automated data import (Internal & Broker files)
* Dynamic column mapping
* Trade matching using composite keys
* Exception detection:
  
      Quantity Break
    
      Price Break
    
      Currency Break
    
      Missing in Broker
    
      Missing in Internal
* Reconciliation dashboard (KPIs + charts)
* Outlook email automation for report distribution
## Tech Stack
* Excel VBA
* Object-Oriented Programming (Class Modules)
* Scripting.Dictionary
* Outlook Automation
## Dashboard
The tool generates a visual dashboard including:
* Match rate
* Break distribution
* Exception summary charts
## How to Use
1. Import Internal and Broker files
2. Run reconciliation macro
3. Generate dashboard
4. Send email report
